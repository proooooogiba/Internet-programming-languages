Коды ответа HTTP

Код ответа (состояния) HTTP показывает, был ли успешно выполнен определённый HTTP
запрос. Коды сгруппированы в 5 классов:

  1. Информационные 100-199
  2. Успешные 200-299
  3. Перенаправления 300-399
  4. Клиентские ошибки 400-499
  5. Серверные ошибки 500-599

Коды состояния определены в 10-ой секции RFC 2616. Обновленную спецификацию можно найти в RFC 7231.


Ниже представлены серверные ошибки:

  • 500 Internal Server Error
  • 501 Not Implemented
  • 502 Bad Gateway
  • 503 Service Unavailable
  • 504 Gateway Timeout
  • 505 HTTP Version Not Supported

Если вы получили код ответа (состояния), которого нет в данном списке, в таком случае он
является не стандартизированным кодом ответа (состояния), вероятней всего он кастомный
сервера.
